何为虚拟节点？

简而言之，虚拟节点即为如下形式：
{ sel: "div", data: { }, elm: **DOMNODE**, key: "div", children: [], text: "" }
sel为 Dom 节点的tag，data为 Dom 节点的props，elm为 Dom 节点的Dom_Node，key为 Dom 节点的唯一标识符，children为 Dom 节点的children_Node，text为 Dom 节点的内容。

Diff算法核心：newNode.children和oldNode.children进行更新

Diff算法的创始人使用了一种巧妙的思想：四指针，并按照程序员平时的思维习惯，提出了四种情况。

  • oldStartNode == newStartNode：直接进行替换；
  • oldEndNode == newEndNode：直接进行替换；
  • newEndNode == oldStartNode：移动oldStartNode到oldEndNode之后；
  • newStartNode == oldEndNode：移动oldEndNode到oldStartNode之前。
